
ういろう - UIROU -


「ういろう」とは。
米粉などに砂糖を練り合わせ、蒸して作った和菓子で、
名古屋銘菓のひとつでもある。


・その『ういろう』がゲームで大活躍?!

舞台はどこかずっと遠くの惑星。そこへ、白ういろうが黒ういろうの追っ手から逃れてやってきます。

白ういろうを狙って次々と飛来する黒ういろうたち。このままでは白ういろうが連れ去られてしまいます!!

あなたは、唯一の対抗手段である『ジャンプアタック』を駆使し、黒ういろうたちを駆逐しなければなりません。


・フル3Dで構築されたモノトーンのちょっと不思議な世界観

ゲーム開始早々、ぽよんぽよんとした『ういろう』独特の柔らかい表現があなたを和ませてくれます。

ですがゲームの舞台はさほど大きくない球体。360度どの方向からも黒ういろうたちが襲ってくる緊迫した状況です。

ちょっとでも気を許すと、たちまち白ういろうは黒ういろうたちに囲まれ、連れ去られてしまいます。


・思わずニヤリとしてしまうご当地ネタ

ゲームの結果画面に表示される『ランク』、実は名古屋の「青柳ういろう」「大須ういろ」が
由来となっていたりします。


操作
攻撃           惑星上で左クリック          
惑星を回転     左ドラッグ                  
惑星を拡大縮小 SHIFTを押しながら左ドラッグ
